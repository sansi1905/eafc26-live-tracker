EA FC 26 Live Tracker - Vercel Yayın

1) Bu klasördeki dosyaları GitHub reposuna yükle.
2) vercel.com hesabı aç.
3) Add New Project > GitHub reposunu seç.
4) Framework: Other / Static
5) Deploy de.
6) Verilen linki Ramazan'a at.

Önemli:
- config.js içinde Supabase URL + Publishable key hazır.
- Secret key yok, koyma.
- Supabase SQL kurulumu zaten yapıldıysa tekrar gerek yok.
- Site açılınca veriler Supabase'den otomatik yüklenir.
